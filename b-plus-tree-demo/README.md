# Java B+树实现 - 学习版

这是一个用Java实现的简易B+树，带有详细的中文注释，便于学习和理解B+树数据结构的原理。

## 项目结构

```
src/main/java/com/bplustree/
├── BPlusTreeNode.java      # B+树节点类
├── BPlusTree.java          # B+树核心实现类
└── BPlusTreeDemo.java      # 演示和测试类
```

## B+树简介

B+树是一种自平衡的多路搜索树，常用于数据库和文件系统的索引结构。

### 主要特点

1. **所有数据都存储在叶子节点中** - 非叶子节点只起到索引作用
2. **叶子节点之间通过指针连接** - 支持高效的范围查询和顺序访问
3. **树的高度保持平衡** - 所有叶子节点都在同一层
4. **节点容量固定** - 每个节点最多存储 `order-1` 个键

## 代码说明

### BPlusTreeNode（节点类）

表示B+树中的一个节点，包含：
- `isLeaf`: 是否为叶子节点
- `keys`: 存储的键列表
- `children`: 子节点列表（仅非叶子节点）
- `nextLeaf`: 指向下一个叶子节点的指针（仅叶子节点）

### BPlusTree（树类）

实现了B+树的核心操作：
- `insert(key)`: 插入键值
- `delete(key)`: 删除键值（支持借位和合并）
- `search(key)`: 查找键值
- `getHeight()`: 获取树的高度
- `getAllKeys()`: 获取所有有序的键
- `printTree()`: 打印树结构

### BPlusTreeDemo（演示类）

演示B+树的各种操作和特性。

## 学习要点

### 1. 插入操作

B+树的插入操作总是在叶子节点中进行：
1. 找到应该插入的叶子节点
2. 在叶子节点中按顺序插入键
3. 如果节点满了（超过 `order-1` 个键），进行分裂操作
4. 分裂时，将中间的键提升到父节点

### 2. 查找操作

从根节点开始，逐层向下查找：
1. 在非叶子节点中找到合适的子节点
2. 移动到对应的子节点
3. 重复直到叶子节点
4. 在叶子节点中查找目标键

### 3. 节点分裂与合并

当节点的键数量超过限制时：
- **叶子节点分裂**：将后半部分移到新节点，中间键提升到父节点
- **内部节点分裂**：中间键提升到父节点，左右部分分别保留

当删除导致节点键数太少时：
- **从左兄弟借位**：借最大的键，更新父节点分隔键
- **从右兄弟借位**：借最小的键，更新父节点分隔键  
- **合并节点**：当无法借位时，合并相邻节点并删除父节点的分隔键

### 4. 叶子节点链表

所有叶子节点通过 `nextLeaf` 指针连接，支持：
- 范围查询（如：查找 10 到 30 之间的所有数据）
- 顺序遍历（如：按顺序获取所有数据）
